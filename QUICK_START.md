# 🚀 Guia Rápido - Painel Flutuante

## 📍 Localização
- **Botão**: Canto inferior direito (📝)
- **Acesso**: Disponível em qualquer tela do sistema

---

## 🎯 Atalhos Rápidos

### Abrir Painel
Clique no botão 📝 no canto inferior direito

### Fechar Painel
- Clique no botão X no topo
- Clique fora do painel
- Clique no botão 📝 novamente

### Mudar de Aba
Clique nos ícones no topo do painel
- 📝 = Anotações
- 🎲 = Dados

---

## 📝 Anotações

### Guardar Notas
1. Abra o painel (clique em 📝)
2. Clique na aba de Anotações
3. Digite o que quiser
4. **Salva automaticamente!**

### Limpar Anotações
1. Clique em "Limpar"
2. Confirme a ação
3. Pronto!

### Exportar Anotações
1. Clique em "Exportar TXT"
2. Arquivo baixa como `anotacoes-sessao-YYYY-MM-DD.txt`
3. Abra em qualquer editor de texto

---

## 🎲 Dados

### Rolar Um Dado
1. Clique na aba de Dados
2. Clique no botão do dado (d4, d6, d8, d10, d12, d20)
3. Veja a animação
4. Resultado aparece em "Resultado: X"

### Rolar Múltiplos Dados
1. Clique na aba de Dados
2. Altere "Quantidade:" para 2-10
3. Clique no dado
4. Vê o resultado total

### Ver Histórico
1. Role para baixo na aba de Dados
2. Veja "Histórico" com todas as rolagens
3. Mostra: tipo de dado, resultado e horário

### Limpar Histórico
1. Clique no ícone de lixo no histórico
2. Confirme
3. Pronto!

---

## 💾 Dados Salvos

### Onde Fica Guardado?
- ✅ **Anotações**: No seu computador (localStorage)
- ✅ **Histórico**: No seu computador (localStorage)
- ✅ **Supabase**: Se estiver logado, sincroniza na nuvem

### Será Apagado Se...
- [ ] ❌ Fechar o painel (não apaga!)
- [ ] ❌ Trocar de tela (não apaga!)
- [ ] ❌ Recarregar a página (não apaga!)
- [ ] ✅ **Apenas** se você clicar em "Limpar"

## ❓ Perguntas Comuns

**P: As anotações sumem ao fechar?**
R: Não! Elas são salvas no seu computador.

**P: Posso compartilhar anotações?**
R: Anotações estão no seu localStorage. Se estiver logado, sincroniza com Supabase.

**P: Quantas rolagens ficam no histórico?**
R: 50 últimas rolagens. Mais antigas são removidas automaticamente.

**P: Como faço backup das anotações?**
R: Use o botão "Exportar TXT" - cria um arquivo baixado.

**P: Onde fica guardado os dados?**
R: Seu computador (localStorage) e Supabase se estiver logado.

**P: Posso usar em outro computador?**
R: Sim! Se estiver logado no Supabase, as anotações sincronizam.

## 🆘 Problemas?

### Painel não abre
- Verifique se há Internet
- Tente recarregar a página
- Teste em outro navegador

### Anotações não salvam
- Verifique se localStorage está habilitado
- Verifique espaço em disco

### Dados não rolam
- Tente recarregar a página
- Verifique console (F12 → Console)

### Histórico vazio
- Já foi criado? Precisa rolar dados primeiro
- Verifi limite de 50 itens
